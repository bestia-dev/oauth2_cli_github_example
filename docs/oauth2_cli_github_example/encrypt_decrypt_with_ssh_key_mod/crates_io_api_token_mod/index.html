Publish to crates.io needs the crates.io secret access_token. This is a secret important just like a password or even more.
There is the original ‚Äúcargo login‚Äù function that saves this critical secret in plain text in ~/.cargo/credentials.toml. Plain-text for secrets in a well-known file is a big no-no. Every malware will just upload it in a millisecond.

I don‚Äôt want to pass secret to an ‚Äúobscure‚Äù library crate that is difficult to review and can change in any point in time to become malicious.

Instead of that, copy and paste this module crates_io_api_token_mod.rs file into your project.
The secrets will stay in your codebase that is easy to inspect and guaranteed that will never change without your consent.
